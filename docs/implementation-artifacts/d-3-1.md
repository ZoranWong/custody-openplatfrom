# Story d-3-1: 存储集成 (Storage Integration)

Status: done

## Story

As a 系统工程师,
I want 将MySQL存储集成到应用架构中,
so that 应用可以使用MySQL作为主要数据存储后端。

## Acceptance Criteria

1. [AC-1] 环境变量 `STORAGE_DRIVER=mysql` 配置后应用使用MySQL存储
2. [AC-2] 所有Repository通过Factory正确获取MySQL Storage Adapter
3. [AC-3] 数据库连接在应用启动时初始化
4. [AC-4] 数据库连接在应用关闭时正确断开
5. [AC-5] 健康检查接口包含数据库连接状态

- MySQL Storage Adapter: `src/repositories/mysql-storage.adapter.ts`

**需要修改/集成的文件：**
- 应用入口: `src/index.ts` 或 `src/app.ts` (添加数据库初始化)
- 健康检查: `src/routes/health.ts` (添加数据库状态)

**环境变量：**
```bash
# .env 文件配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=cregis_openplatform
DB_USER=root
DB_PASSWORD=
STORAGE_DRIVER=mysql
STORAGE_TYPE=mysql
DATABASE_URL=mysql://root@localhost:3306/cregis_openplatform
```
