## Dev Notes

### Technical Background

JSON序列化工具确保客户端和服务器端对同一数据产生相同的JSON字符串，从而保证MD5哈希值一致。

**序列化规则:**
1. Key按字母顺序排序（递归处理嵌套对象）
2. Number类型保持原样
3. Date类型转换为ISO字符串
4. null值可配置是否包含
5. undefined值默认移除
