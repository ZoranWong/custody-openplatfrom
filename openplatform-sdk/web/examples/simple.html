Cregis OpenPlatform SDK 示例

配置信息

App ID: app_d871206a-7477-4c
Auth URL: http://api.vaulink.com/openplatform/oauth/
App Token: sk_d871206a74774ca4890d57eb0ddff009

这是一个简单的内联授权示例。appToken 应从后端接口获取，SDK 通过 URL 参数传递 appToken 和用户 token，授权完成后结果会显示在下方。

注意：appToken 是应用密钥，请勿暴露在前端代码中。必须从后端接口获取 appToken 后再传递给 SDK。
后端生成算法：md5(appId + appSecret + timestamp + nonce) + '-' + timestamp + '-' + nonce
